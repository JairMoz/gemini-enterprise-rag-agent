# Company Technical Support Manual (Knowledge Base)

## Connectivity Issues
Users experiencing "Error 404" should first verify their API keys in the dashboard. If the error persists, ensure the region is set to `us-central1`.

## Refund Policy
Refunds are processed within 5-7 business days. Users must provide a transaction hash for all claims.

## Security Protocols
Multi-factor authentication (MFA) is mandatory for all administrative accounts. Agents should never ask for a user's password.
